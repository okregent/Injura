# Injura — Product Vision

## Overview
Injura is an adaptive AI-powered squat analysis platform designed to provide personalised movement feedback using pose estimation and biomechanics-aware logic. Unlike generic fitness applications that enforce a single "perfect squat form", Injura adjusts movement feedback based on body proportions, mobility limitations, and user discomfort areas.

## Core Problem
Most existing AI fitness applications rely on fixed movement thresholds and generic coaching rules. However, squat mechanics vary significantly depending on factors such as femur length, torso length, ankle mobility, and injury history. Injura provides adaptive movement analysis instead of one-size-fits-all correction.

## Key Differentiator
Injura does not attempt to force a single "ideal squat form". Instead, it adjusts movement thresholds according to body proportions, mobility limitations, and user discomfort inputs.

## Scalability Philosophy
Injura is designed as a production-oriented AI system rather than a short-term portfolio project. Scalability, maintainability, modularity, and future expansion are core design priorities.

## Reliability Over Complexity
The project prioritises:
- Reliable movement analysis
- Stable landmark detection
- Explainable feedback
- User trust

Practical usability is prioritised over experimental AI complexity.

## Long-Term Vision
The long-term goal is to evolve Injura into an adaptive human movement intelligence platform capable of supporting:
- Strength training analysis
- Combat sports analysis
- Mobility assessment
- Ergonomics analysis
- Rehabilitation assistance
- Personalised movement intelligence systems
